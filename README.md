# AvaloniaProject
This is a sample Avalonia application that utilizes several frameworks and libraries, and can serve as a reference when creating new projects.

- Used the [ReactiveUI](https://github.com/reactiveui/ReactiveUI) MVVM framework.
- Used the [Semi.Avalonia](https://github.com/irihitech/Semi.Avalonia) and [Ursa.Avalonia](https://github.com/irihitech/Ursa.Avalonia) themes.
- Used the [Icons.Avalonia](https://github.com/Projektanker/Icons.Avalonia)'s Material Design icons.
- Published using AOT.
